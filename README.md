## Requirement
- npm v8.19.2

## Installation
- Install node modules Tailwind CSS
  ```bash
  npm install -D tailwindcss
  ```

- Start the Tailwind CLI build process
  ```bash
  npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
  ```
